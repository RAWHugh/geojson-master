# 在线绘制GeoJSON需求文档

## 技术要求
- 前端框架： vue/vite
- 地图API：Openlayers
- 底图图层：高德底图/天地图地图/天地图影像底图

地图服务[底图服务](https://www.cnblogs.com/wongjzzz/p/16480208.html)。

## 需求细节

### 一、页面整体概况
参考 **geojson.io** 在线地理空间数据编辑工具页面，由Mapbox提供地图底图引擎，是免费网页端GIS轻量工具，无需安装软件，专门用来创建、编辑、转换、预览GeoJSON/TopoJSON/KML等空间矢量数据。页面采用**三栏经典布局**：左侧绘图操作栏、中间交互式地图画布、右侧数据代码/表格面板。


### 二、分区域拆解功能
#### 1. 左上角：绘图工具栏（矢量绘制核心）
一排图标是地理要素绘制工具，从左到右：
1. **选择移动工具（蓝色箭头）**：选中已绘制的点/线/面，拖拽调整坐标、顶点位置
2. **点标记工具**：单击地图生成点位（POI、采样点、坐标标记）
3. **折线工具**：连续点击绘制路径、管线、路线等线要素
4. **多边形工具**：闭合点击绘制地块、行政区、范围面；附带矩形快速框选、圆形圈选快捷按钮
5. 附属小控件：`Label`标签下拉（控制要素文字标注显示）、定位居中、清空选中要素等辅助按钮

#### 2. 左侧边栏：导入与引导区
- 引导文字：提示新手「用上方绘图工具画要素，或导入空间文件」
- **+ Import 导入按钮**：支持拖拽/上传多格式空间文件：GeoJSON、TopoJSON、KML、CSV（带经纬度转点位）、Shapefile、GPX等格式，一键加载到地图画布
- `Supported file types`：展开查看完整支持文件格式列表

#### 3. 中间核心：Mapbox交互式地图画布
- 底图：当前是浅灰色简约街道底图，底部可切换卫星、深色、浅色、OSM等多种底图样式
- 地图控件（画布左下角）：
  - 放大/缩小加减按钮
  - 上下左右平移快捷箭头
  - 定位图标：一键回到当前视图中心点
- 画布自带搜索框（右上角Search，快捷键Ctrl+K）：输入地名/地址快速跳转定位（图中已定位到湖南株洲云田片区）
- 比例尺：右下角显示距离标尺（当前2英里长度参照）
- 联动特性：地图上每画一个要素，右侧JSON代码会**实时同步更新**；修改右侧代码，地图图形也立刻刷新双向联动

#### 4. 右上角全局操作区
- `powered by mapbox`：地图引擎标识；右侧「Sign up for Mapbox」是Mapbox平台注册入口
- `Import / Export`：全局导入导出菜单：
  - Import：同左侧上传功能
  - Export：导出为GeoJSON、KML、CSV、Shapefile、TopoJSON等十几种格式，实现空间格式互转

#### 5. 右侧大面板：数据编辑双视图（JSON / Table切换）
1. **JSON标签（当前激活）**
    实时展示标准GeoJSON源码，初始是空模板：`{"type":"FeatureCollection","features":[]}`
    支持手动手写修改坐标、properties属性、几何类型，改完地图实时渲染；可全选复制代码直接粘贴到项目、前端地图框架（Leaflet、Mapbox GL、OpenLayers）使用。
2. **Table表格标签**
    切换成表格视图，以行列形式批量编辑所有要素的属性字段（名称、编号、自定义业务字段），单元格修改和地图图形双向同步，适合批量录入业务属性信息。

## 实现需求：
1. **零门槛绘图**：鼠标拖拽画点线面，不用专业GIS软件（ArcGIS/QGIS）
2. **双向实时联动**：地图可视化 ↔ GeoJSON源码/属性表格 同步互改
3. **多格式互转**：导入KML/SHP/CSV，导出GeoJSON，解决不同平台空间数据兼容问题
4. **属性完整编辑**：既改几何坐标，也能给每个地块/点位加名称、备注、分类等业务属性
5. **轻量无安装**：浏览器直接打开即用，支持复制代码、下载文件、生成分享链接协作编辑
